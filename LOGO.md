# Strategickhaos Trademark Logo

## Overview
This directory contains the official Strategickhaos trademark logo files for use with GitHub OAuth and other branding purposes.

## Files
- **strategickhaos-logo.png** - PNG format with transparency (11KB, 500x500px) - **Recommended for GitHub OAuth**
- **strategickhaos-logo.jpg** - JPEG format with solid background (48KB, 500x500px) - Alternative format

## Design Elements

### Visual Components
1. **Toruk Silhouette** - Great Leonopteryx (Ikran Makto) at center
   - Central diamond-shaped body in toruk orange
   - Swept-back triangular wings
   - Distinctive tail and head crest
   
2. **Orbital Rings** - Representing Node 137 satellite
   - Primary orbital ring in cyan
   - Secondary tilted elliptical orbit in orange
   - Node 137 marker at 137° position

3. **Sacred Geometry** - Ratio Ex Nihilo pattern
   - Golden ratio (φ = 1.618) based concentric circles
   - Spiral approximation using golden ratio relationships
   - Subtle background layer

4. **Color Scheme**
   - **Bio-luminescent Cyan**: `#00FFFF` (RGB: 0, 255, 255)
   - **Toruk Orange**: `#FF8C00` (RGB: 255, 140, 0)
   - **Dark Background**: `#0F0F14` (RGB: 15, 15, 20)

### Corner Accents
- Stylized corner brackets in alternating cyan and orange
- Represents "SKH" branding and technical/systematic approach

## Usage

### GitHub OAuth Application
1. Navigate to GitHub Settings → Developer settings → OAuth Apps
2. Create new OAuth App or edit existing
3. Upload `strategickhaos-logo.png` as the application logo
4. The 500x500px size meets GitHub's requirements

### Other Uses
- Social media profile pictures
- Documentation headers
- Product branding
- NFT/trademark representation

## Technical Specifications
- **Dimensions**: 500 x 500 pixels (square)
- **Aspect Ratio**: 1:1
- **Color Mode**: RGBA (PNG) / RGB (JPG)
